# Movies_Chanllenge

## Documentation

https://documenter.getpostman.com/view/19298430/2sAYBYgWMK


## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

```

## Run tests

```bash
# unit tests
$ npm run test


# test coverage
$ npm run test:cov
```
## Description
<p>O projeto foi feito utilizando nestjs, com typeorm e postgresSQL, o postgres eu rodei no docker, pode usar como preferir nessa parte. O projeto falta alguns detalhes para ficar 100%</p>
